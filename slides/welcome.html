Grundlagen

HTTP

HTTP wird zur Kommunikation zwischen Client und Server genutzt. Typi- scher Ablauf der Client- Serverkommunikation:

  • Der Client (z.B. Internetbrowser) sendet eine Anfrage an den Server.
  • Der Server empfängt die Anfrage und bearbeitet sie. .
  • Der Server sendet eine Antwort
  • Der Client erhält die Antwort

Ajax

Bei der klassischen Client- Serverkommunikation werden Daten, meist durch abschicken eines HTML-Formulars, synchron zum Server übertragen, was gleichzeitig ein neu laden der Website auslöst. Der Client verharrt in der La- deposition, bis er eine Antwort des Severs erhält. Bei AJAX hingegen werden Anfragen über ein XMLHttpRequest JavaScript-Object ausgeführt, was eine asynchrone Kommunikation möglich macht.
